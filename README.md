# Big Data Management & Analytics - University Project

This repository contains the R scripts developed for the Big Data Management and Analytics 
individual assignments at Rotterdam School of Management (MSc Business Information Management 25/26).

## Project Overview
The tasks comprised: cleaning, combining, and analysing integrated datasets; 
performing exploratory data analysis, linear regressions, logistics regressions;
building decision trees.
* **Data Cleaning**
* **Combining and analysing integrated datasets**
* **Exploratory Data Analysis (EDA):**
* **Statistical Modeling**

## Technologies Used
* **Language:** R
* **Libraries:** ggplot2, dplyr, readr, plotrix, stargazer, viridis, arules, pROC

## How to Run
1. Clone the repository.
2. Open the `.R` files in RStudio.
3. Ensure the dataset is located in the `/data` folder (if applicable).
